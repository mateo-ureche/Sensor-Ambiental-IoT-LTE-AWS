# Dashboard Web

Dashboard interactivo desarrollado para la visualización remota de variables ambientales transmitidas desde un nodo IoT basado en ESP32-S3 y SIM7670G.

## Funcionalidades

* Visualización en tiempo real de temperatura, humedad y presión atmosférica.
* Integración GPS con visualización cartográfica mediante Leaflet.
* Cálculo automático de índice de calor (Heat Index).
* Cálculo automático de punto de rocío (Dew Point).
* Sistema de alertas visuales.
* Predicción básica mediante regresión lineal.
* Generación de reportes PDF.
* Exportación de datos en formato CSV.
* Gráficas históricas interactivas.
* Comparación estadística diaria.
* Monitoreo de estado del dispositivo.

Tecnologías utilizadas:

* HTML5
* CSS3
* JavaScript
* Chart.js
* Leaflet.js
* jsPDF
* AWS API Gateway
* AWS Lambda
* Amazon DynamoDB
