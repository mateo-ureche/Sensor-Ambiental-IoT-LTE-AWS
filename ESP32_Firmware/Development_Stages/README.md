# Development Stages

Esta carpeta contiene los programas de prueba desarrollados durante las primeras etapas de integración del sistema IoT.

Cada archivo corresponde a una fase específica de validación de hardware antes de la construcción del firmware final.

## Contenido

### Stage_01_Serial_Test
Verificación básica de funcionamiento del ESP32-S3 y comunicación serial.

### Stage_02_DHT22_Test
Prueba de lectura de temperatura y humedad mediante sensor DHT22.

### Stage_03_BMP280_Test
Prueba de adquisición de presión atmosférica y temperatura utilizando BMP280.

### Stage_04_RGB_LED_Test
Validación del LED RGB WS2812B integrado para indicación visual de estados.
